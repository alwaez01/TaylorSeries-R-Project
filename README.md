# How does a computer calculate sin(x)? 

Have you ever wondered what happens "under the hood" when you call a trigonometric function in your code?

This project explores how computers calculate $\sin(x)$. It provides a detailed analysis of how the 
Maclaurin series (a special type of Taylor series centered at zero) is used to approximate various 
functions using simple polynomials.
